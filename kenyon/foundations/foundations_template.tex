\begin{problem1}{2} Let $A$, $B$, and $C$ be sets. Then $A\times (B\cup C)=(A\times B)\cup(A\times C).$\\
\begin{proof}
\indent Let $A,B,$ and $C$ be sets, and let $(x,y)\in(B\cup C)$. Then $x\in A$ and $y\in B\cup C$, which means $y\in B$ or $y\in C$. If $y\in B$, then $(x,y)\in A\times B$, so certainly $(x,y)\in (A\times B) \cup (A\times C)$. Similarly if $y\in C$, then $(x,y)\in A\times C$, so certainly $(x,y)\in (A\times B) \cup (A\times C)$. Therefore $A\times (B\cup C)\subseteq(A\times B)\cup(A\times C).$

Let $(x,y)\in (A\times B)\cup(A\times C)$. Then $(x,y)\in (A\times B)$ or $x\in(A\times C)$.
\end{proof}
\end{problem1}
